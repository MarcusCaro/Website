Welcome to my Portfolio!

Fruit Catcher (1.1.9)

My first project was a game called Fruit Catcher.
Fruit would fall from the sky and the player would have to catch the fruit with the basket to earn points.
We used turtles for the fruits and the basket while using while loops in order to detect collisions and keep the fruit falling from the sky.

Temple of the Duck (1.2.5)

My second project is a game called Temple of the Duck.
The player controls a duck using the arrow keys to turn and the "a" and "d" keys to change speed.
Bread and spikes will randomly generate within the borders. Collecting bread will earn a point while collecting a spike will end the game.
We used list indexing for a leaderboard as well.

Super Mario (Scratch)

My third project is a knockoff of Super Mario Bros.
The player can use the arrow keys to navigate mario across the screen and onto platforms.
The player needs to collect coins while avoiding the goombas.
The game will end if the timer reaches 0 or if the player touches a goomba.

A pHishy Fish Tank (2.1.6)

My fourth project revolved around the security concerns of a company's program and how to fix it.
A security breach occurred through a phishing email. The company's program was tampered with.
Using debugging techniques, we were able to fix the program and suggest some improvements to be made.
